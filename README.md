# Pensamento-Computacional-Gamificado

1- Plataforma Web Educacional que utiliza os pilares do pensamento computacional de forma gamificada, seções explicando os pilares do pensamento computacional e como esses pilares se relacionam a um jogo desenvolvido na unity que estará dentro da aplicação Web.
2- O público-alvo são alunos e professores do ensino fundamental 2 da rede pública de ensino, para aluno sendo a idade de 11 anos ou mais.
3- Platarforma educacional, que ultiza a gamificação para ensinar os pilares de pensamento computacional, e permitindo que professores sem conhecimento prévio em progamação possam aplicá-las.	
4- O usuário irá desenvolver conhecimento teórico e partico dos pilares do pensamento computacional.
